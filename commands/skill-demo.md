# /skill-demo

Package the research workflow as a live demo.

## Inputs

- Demo question
- Target challenge or audience
- Expected demo length

## Steps

1. Show messy question.
2. Show decomposition.
3. Show source map.
4. Show triangulated claim.
5. Show findings grid.
6. Show thesis and caveats.
7. Show how the output becomes reusable.

## Output

- Demo script
- Demo prompt
- Expected result
- Submission summary
