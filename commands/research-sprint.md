# /research-sprint

Run the full onchain research workflow from messy question to thesis.

## Inputs

- Research question
- Decision it informs
- Deadline
- Expected format

## Steps

1. Intake
2. Decompose
3. Assign source categories
4. Triangulate highest-stakes claim
5. Build findings grid
6. Synthesize thesis
7. Produce requested artifact

## Output

- Research analysis
- Findings grid
- Thesis
- Caveats
- Source list
