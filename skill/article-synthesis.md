# Article Synthesis

Use when turning findings into an article, memo, or X thread.

## Thesis Shape

Write the thesis as:

> [Actor/market] is shifting from [old bottleneck] to [new bottleneck], because [evidence], but [caveat].

Example:

> Mantle's RWA push is less about listing tokenized assets and more about building the distribution, execution, liquidity, and agent infrastructure those assets need to become usable markets.

## Article Outline

1. Hook: start with a concrete recent market move.
2. Thesis: state the position clearly.
3. Evidence: show the numbers and source categories.
4. Mechanism: explain why the move happened.
5. Landscape: compare what makes this actor different.
6. Caveat: name the strongest counterargument.
7. Next move: make a falsifiable call about what comes next.

## X Thread Format

Use 7-10 posts:

1. Thesis
2. Why now
3. Key data
4. Mechanism
5. Competitive distinction
6. Caveat
7. What comes next
8. Source note or chart note

## Required Ending

Include:

- Caveats and uncertainty
- What would change my mind
- Source list

Avoid hype language when the evidence is only medium confidence.
