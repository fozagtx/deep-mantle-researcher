# Evidence Grid

Use for findings, confidence, and contradiction tracking.

## Grid Columns

| Sub-question | Source(s) | Finding | Confidence | Contradictions or gaps |
|---|---|---|---|---|

## Confidence Scoring

High:

- Recent primary evidence
- Independent corroboration
- Clear definitions
- No unresolved contradiction

Medium:

- One strong source
- Multiple interested sources that agree
- Recent but not independently verified
- Clear caveats

Low:

- Stale data
- Indirect evidence
- Ambiguous definitions
- Contradicted sources
- Incentive-heavy claim without corroboration

## Required Caveat Types

For onchain finance, check whether the finding depends on:

- Token holder rights
- Jurisdictional eligibility
- Custody or issuer risk
- Liquidity incentives
- Dashboard methodology
- Market hours versus 24/7 trading
- Oracle or RFQ pricing quality
- Redemption and settlement path

## Contradiction Handling

If sources disagree:

1. Name the contradiction.
2. Prefer primary mechanics over marketing language.
3. Prefer direct data over summaries.
4. Lower confidence until resolved.
5. State what evidence would resolve it.
